Python libraries for research data analysis

These libraries are made to ease the work of data analysis on Particle Image Velocimetry (PIV) data. Also, to get rid of finding unique name for every project, I have added "ddr" as prefix to simple logical project name.

  • ddr_davis_data :- Contains classes and functions for accessing and plotting .vc7 and .im7 files obtained from Davis software.
  • ddr_analysis_tools:- Contains classes for performing analysis such as Proper Orthogonal Decomposition (POD), Spectral Proper Orthogonal Decomposition (SPOD), Fast Fourier Transform (FFT), Short-Term Fourier Transform (STFT) etc.
  • ddr_mfc:- This package helps in controlling Alicat make mass flow controllers (MFC) from python. This helps in integrating the MFC controll in Tkinter environment.
  • ddr_cantera:- Helping functions for calculations of air properties at different temperatures and pressures. For example, this was used in obtaining the Mach number of the flow at given flowrate and pressures at any given cross-section of the flow path.
